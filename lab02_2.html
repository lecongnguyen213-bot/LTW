Yêu cầu thực hành danh sách

Ordered List(Danh sách có thứ tự)

Hãy liệt kê các bước để thực hiệ công việc nào đó:

  1. Bước 1:...
  2. Bước 2:...
  3. Bước 3:...
  4. Bước 4:...
  5. Bước 5:...

Urdered List(Danh sách không có thứ tự)

Hãy liệt kê 3 lợi ích của học tập chăm chỉ:

  • Lợi ích 1:...
  • Lợi ích 2:...
  • Lợi ích 3:...

Description List(Danh sách mô tả)

Hãy liệt kê 3 lợi ích của học tập chăm chỉ:

Python
Ngôn ngữ lập trình dễ và linh hoạt
JavaScript
Ngôn ngữ dùng để xây dựng các Web động
Java
Ngôn ngữ lập trình phổ biến , chủ yếu được ứng dụng trong phát triển doanh nghiệp
